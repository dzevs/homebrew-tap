# homebrew-tap

Homebrew tap for [zynk](https://github.com/dzevs/zynk) — a terminal-native command center for AI agents.

## Install

```sh
brew install dzevs/tap/zynk
```

or:

```sh
brew tap dzevs/tap
brew install zynk
```

The `zynk` formula installs the prebuilt binary from the
[zynk GitHub Releases](https://github.com/dzevs/zynk/releases) (macOS arm64/x86_64, Linux arm64/x86_64 —
glibc ≥ 2.30). The binaries are not code-signed/notarized; see the formula caveats.
